Class: Matrix

Matrix()

The Matrix class represents an "n" by "m" dimensional vector Dependecies: - misc.js - vector.js

Constructor

new Matrix()

Creates a Matrix. The arguments can be: - Nothing: The Matrix will have 0 rows and 0 columns - One Matrix: In this case this is a copy constructor - One Vector, bool: The matrix will represent the Vector. if the bool argument is ommited or false the vector is treated as a row vector, if the bool === true the vector is treated as a column vector - A series of Objectswhiches can define a Vector: Creates Vectors from the Objects, and then creates a Matrix of those Vectors - An Array of the above: Works as the above - A JSON string of the above works the same - arguments[0] == 'init': arguments[1] an array where arguments[1][0] is the number of rows and the arguments[1][1] is the number of columns of this matrix (every dimension's vaule is arguments[2] or 0 if omitted) - arguments[0] an integer: arguments[0] as the number of columns then one number parameter for every element. The given numbers will be the element values from left to right then from top to bottom (the rest will be == 0)
